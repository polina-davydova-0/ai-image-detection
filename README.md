# Исследование методов обнаружения AI-изображений

## Описание проекта

В рамках научно-исследовательской работы (НИР) проведено исследование методов распознавания изображений, созданных генеративным искусственным интеллектом.

**Цель:** оценить способность существующих детекторов распознавать изображения от разных типов генераторов (GAN, диффузионные модели, VAE, гибридные архитектуры)

**Автор:** Давыдова Полина Алексеевна (группа 6303-010302D)

## Использованные детекторы

| Детектор | Архитектура |
|----------|-------------|
| AIorNot-SigLIP2 | SigLIP2 (ViT, 86M параметров) |
| AI-image-detector | Swin Transformer (27M параметров) |
| AI-vs-Deepfake-vs-Real-Siglip2 | SigLIP2 (86M параметров) |

Подробнее о загрузке моделей см. в `models/README.md`

## Основные результаты

- Создана сбалансированная БД из 260 изображений (130 real + 130 fake)
- Протестировано 3 детектора на своей БД и датасете Tiny GenImage
- Лучший детектор (AI-vs-Deepfake) показал точность **67.69%** на своей БД
- На датасете Tiny GenImage точность варьируется от 37% до 53% в зависимости от типа генератора

## Как воспроизвести эксперимент

1. Скачайте базу данных по ссылке из `LINKS.md`
2. Установите зависимости: `pip install transformers torch sklearn tqdm pillow`
3. Откройте ноутбуки в папке `notebooks/` в Google Colab или локально
